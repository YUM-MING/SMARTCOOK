# 🧊 **SMARTCOOK**
> 한국전자전(KES) 출품 및 코엑스·졸업 전시회 전시작  
> YOLO 기반 재료 인식으로 개인 맞춤형 레시피를 추천하는 AI 웹 서비스


## 📅 **개발 기간**
> 2025.04.07 ~ 2025.11.04  


## 🎯 **프로젝트 목적**
> 최근 통계에 따르면, **청년 1인 가구의 86.8%가 요리에 어려움을 겪고 있으며**,  
> 가정 내 식품 낭비율이 전체 낭비의 절반 이상을 차지하고 있습니다.  
> 또한, 스마트 주방기기의 43% 이상이 ‘가격 부담’을 느끼고 있다는 조사 결과가 있습니다.  
>
> 이러한 문제를 해결하기 위해 **SMARTCOOK**은 다음을 목표로 합니다:
>
> - 📸 **AI 재료 인식**을 통해 복잡한 재료 입력 과정을 자동화하고,  
> - 🧠 **맞춤형 레시피 추천**으로 사용자의 건강·기호·재료 상황을 반영하며,  
> - 🌱 **식재료 낭비를 줄이는 ESG 실천형 플랫폼**으로 발전합니다.  
>
> 즉, **누구나 스마트 냉장고 없이도 AI 주방을 경험**하고  
> **즐겁고 지속 가능한 요리 생활**을 가능하게 하는 것이 프로젝트의 핵심 목적입니다.


## 📖 **프로젝트 소개**
> **SMARTCOOK**은 냉장고 속 재료를 카메라로 촬영하면 YOLO 객체 인식 모델이 자동으로 재료를 감지하고,  
> 이를 기반으로 사용자의 선호, 건강 정보, 재료 보유 상태를 고려해 **맞춤형 레시피를 추천하는 웹사이트**다.  
>  
> 고가의 스마트 냉장고 없이 **일반 스마트폰만으로 AI 주방 경험**을 제공하는 것을 목표로 하며,  
> 사용자 편의성과 식재료 낭비 절감을 동시에 실현한다.


## 🧠 **핵심 기능**
| 기능 | 설명 |
|------|------|
| **AI 재료 인식** | YOLOv8 모델을 활용해 업로드된 음식 이미지를 분석하고 자동으로 재료를 추출 |
| **실시간 레시피 추천** | 인식된 재료를 바탕으로 최적의 요리를 추천하며, 부족한 재료도 함께 제안 |
| **맞춤형 필터링** | 재료 수·관련도·비건 여부·비선호 재료 등을 반영해 개인화된 추천 제공 |
| **음성 기반 인터랙션** | STT/TTS 기능을 활용해 음성으로 명령하고 음성 안내로 레시피를 들을 수 있음 |
